Declaración de Privacidad de Aplicación Móvil Esta política de privacidad se aplica al uso de una aplicación de software ("Aplicación") en un dispositivo móvil que fue creado por PLANEACIÓN DE MEDIOS, S.A. de nombre comercial Proximity Guatemala, para Pollo Campero, Sociedad Anónima (quien a lo largo del texto se identificará indistintamente como “Pollo Campero”), el nombre de la Aplicación es Banquetes Campero la cual permite al usuario hacer pedidos por medio de un correo electrónico desde su dispositivo móvil, seleccionando el tipo de banquete y la combinación de salsas y aderezos que desee. El pago se realiza en persona cuando el repartidor llega a la ubicación del usuario y le entrega su pedido. La Aplicación no solicita ninguna información al usuario más que la necesaria para hacer el contacto y entrega del pedido. No se realiza ninguna transacción monetaria dentro de la Aplicación. ¿Qué información recolecta la aplicación y cómo se utiliza?
La aplicación obtiene la información que el usuario (quien a lo largo del texto se identificará indistintamente como “Usted” o “Su”) proporciona cuando descarga y se registra a la aplicación. Registrarse es necesario para poder utilizar las funciones que ofrece la aplicación.

También se podrá utilizar la información que Usted ha facilitado para comunicarnos con usted en algún momento y ofrecerle información importante y promociones de marketing.
